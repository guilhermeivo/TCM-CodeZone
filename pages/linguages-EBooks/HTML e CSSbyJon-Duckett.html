HTML e CSS Projete e Construa Websites

by Jon Duckett

Description

Welcome to a better way to learn HTML & CSS. Whether you want to design, build from scratch, or have more control over an existing website, this book will help you create engaging, user-friendly content. We know code can be a little intimidating, but take a look and see how this guide differs from more traditional programming books. All pages feature a new topic and simple, visual way, with straightforward explanations accompanied by code samples. You'll find help on how to organize and design your website pages so you can create great, easy-to-use websites. You do not need experience!


About author

Jon Duckett has been designing and building web sites for over a decade, working with global brands, and has authored and co-authored over a dozen books on web design, programming, usability and accessibility. He has curated conferences about web programming and has been a web strategy consultant, often coding front ends for projects and designing architectures for back end programming. He is the author of the best-selling, HTML & CSS: Design and Build Websites and JavaScript & jQuery: Interactive Front-End Web Development.
